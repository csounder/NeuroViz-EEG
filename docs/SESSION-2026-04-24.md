## How to run locally (tomorrow)

1. **Backend (optional for charts-only / sim):** start `server-enhanced.js` → HTTP 3000, WS 8080.
2. **Frontend:**
   ```bash
   cd web
   npm install   # if needed
   npm run dev   # http://localhost:3001
   ```
3. If **port 3001 busy:**
   ```bash
   lsof -nP -iTCP:3001 -sTCP:LISTEN
   kill <PID>
   ```
4. If **weird chunk/CSS 404s:**
   ```bash
   cd web && rm -rf .next && npm run dev
   ```
   Then hard-reload the browser (or new tab).
